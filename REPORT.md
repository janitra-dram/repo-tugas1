# Buat Laporan di Sini
Program ini mengimplementasikan alur input-output dasar pada C++ dengan memanfaatkan library iostream dan string. Komponen utamanya meliputi penggunaan using namespace std untuk efisiensi penulisan kode, fungsi int main dengan parameter command-line arguments (argc dan argv) sebagai entry point utama, serta deklarasi variabel nama bertipe string untuk alokasi memori teks. Proses pengambilan data dilakukan menggunakan fungsi getline(cin, nama) agar dapat menangkap input secara utuh termasuk karakter spasi, yang kemudian hasilnya diproses melalui objek cout untuk menampilkan sapaan dinamis ke layar sebelum program mengakhiri eksekusi dengan status return 0.
untuk mengubah output dari myapp ke MyApp waktu compile kita harus menentukan nama file nya g++ main.cpp -o MyApp
